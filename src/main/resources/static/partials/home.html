Alignment Portal for genomic sequences

This portal allows you to upload genomic sample sequences and reference genomes sequences. The sample sequences are aligned with reference genome using bowtie and samtools. Variants files are generated using bcftools. Aligned reads and variants files are uploaded to google cloud storage and then imported into google genomics platform.

Submit New Genomic Sequences

Click this button to submit new genomic sequences. This process will take you through the steps of uploading reference genome and sample sequences. These files will then go through several processing steps including generation of alignment files and variant files. Once the files have been uploaded you can monitor their progress by going to the Check Job History link.

Check Job History

Click this button to check job history of jobs being processed. You will be able to see the status of different files and additional information about them. Once the process completes you will see the links to View Reads and View Variants for you files.

Explore Reads

Once the aligned files have been imported into google genomics platform. You can then view the sequence reads in a formatted fashion. You can search for aligned sequences and paginate through returned results and be able to review detailed information about aligned sequences.

Explore Variants

Once the variant files have been imported into google genomics platform. You can then view the variants for your sequence files. You can search for variants and paginate through returned results and be able to review details information about variants.

This portal uses bowtie, samtools, bcftools, google cloud storage, google genomics api and google big query.
Complete source code for this application is available at https://github.com/adeelmahmood/alignmentportal
