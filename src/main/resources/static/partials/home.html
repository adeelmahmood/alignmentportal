Alignment Portal for genomic sequences

This portal allows you to upload genomic sequences and align them with uploaded reference genomes. Once the data is aligned you can explore various aspects of it and interact with the aligned data using different tools provided by this portal. Below are the steps for uploading and processing a new set of genomic sequences

Steps to submit new genomic sequences

  1. Create a new dataset A dataset is a logical group for your sequences
  2. Upload reference sequences fasta files for reference genome sequences
  3. Upload sample sequences fasta files for sample sequences
