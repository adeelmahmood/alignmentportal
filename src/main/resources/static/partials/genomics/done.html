Reference index will be generated
In this step bowtie software is used to generate a reference index from your uploaded reference genome file
Sample file will be aligned using generated reference index
In this step bowtie software is used to align the uploaded sample sequence with uploaded reference genome
Aligned SAM file will be converted to BAM file
The default output format for bowtie aligned files is SAM. In this step SAMTools are used to convert this file into to its binary equivalent BAM.
BAM file will be sorted
In this step binay BAM file is sorted
Using the aligned sorted BAM file a variants will be generated
In this step SAMTools and BCFTools are used to generated a variants file
